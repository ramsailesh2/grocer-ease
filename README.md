# GrocerEase - Online Grocery Shopping App

GrocerEase is a mini grocery shopping web application built using ReactJS.  
It allows users to browse products, search items, manage cart and place orders.

## Features
- Category-based product list
- Search functionality
- Add to cart with quantity control (+ / -)
- Cart management
- Checkout confirmation
- Profile login simulation
- Random featured products on home page
- Professional UI design

## Technologies Used
- ReactJS
- React Router DOM
- CSS
- JSON Server
- Axios
